晓月圆舞曲怪物属性格式:
  长度0x24 字节.
  以下面第一行举例：00为起始字节，4为长度，后面是描述
  00,4: Create code 指针.
  04,4: Update code 指针.
  08,2: Item 1. 道具编辑器里的道具的道具id加1.
  0A,2: Item 2. 道具编辑器里的道具的道具id加1.
  0C,2: 怪物HP上限.
  0E,2: 怪物MP上限.
  10,2: 杀死怪物得到的经验.
  12,1: 魂稀有??
  13,1: 怪物攻击力.
  14,1: 怪物防御力.
  15,1: ???
  16,1: ???
  17,1: 魂类型.
    0 - 红魂.
    1 - 蓝魂.
    2 - 黄魂.
    3 - 青魂.
  18,1: 魂的ID，在道具编辑器对应类型魂中名字前面的id.
  19,1: ???
  1A,2: 怪物的攻击属性.
  1C,2: 怪物的抗性.
  1E,2: ???
  20,2: ???
  22,2: ???
